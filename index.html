Meu nome é Carlos Almeida, sou um desenvolvedor back-end com 3 anos de experiência no desenvolvimento de aplicações utilizando principalmente as tecnologias .NET Core, C#, Node.js, Docker, Azure e React.js.

Neste período tive a oportunidade de trabalhar no desenvolvimento dos mais diversos tipos de produtos, desde aplicações web e mobile monolíticas até aplicações distribuídas orientadas a microsserviços.

A maior parte do tempo estive atuando no back-end com o desenvolvimento de APIs e serviços, porém, também já tive experiências atuando no front-end utilizando React.js.
